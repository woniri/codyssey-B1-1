# Few-shot Examples (최소 3개)

## Few-shot 예시 1: 좋은 입력 → 좋은 출력 (기본 케이스)

**입력:**
[업무 과업] 회의록 요약

[회의 정보]

회의명: 마케팅 팀 주간 미팅
날짜: 2026-06-02
[회의 내용]
신제품 런칭 일정을 7월 15일로 확정했습니다. A팀은 SNS 광고 예산 3천만원을 요청했습니다.

**출력 (예시):**
결정사항
신제품 런칭 일정: 2026년 7월 15일 확정
Action Items
A팀: SNS 광고 예산 3천만원 요청 검토 (기한 미지정)
후속 일정
추가 정보 확인 필요
---

## Few-shot 예시 2: 모호한 입력 → 확인 질문 (중요!)

**입력:**
[업무 과업] 회의록 요약

[회의 내용]
일정이 미뤄졌고 예산도 조정해야 할 것 같아요. 팀원 의견이 갈렸습니다.

**출력 (예시):**
확인 질문:

구체적인 미뤄진 일정과 원래 일정이 어떻게 되나요?
예산 조정 규모와 항목이 무엇인가요?
타협안의 구체적인 내용은 무엇인가요?
위 정보를 확인 후 요약을 진행하겠습니다.

---

## Few-shot 예시 3: 긴 내용 + 조건 추가

**입력:**
[업무 과업] 회의록 요약
[톤] 단호하고 전문적으로
[금지] 추측성 표현

[회의 내용]
... (긴 내용)

**출력 (예시):**
결정사항
...
Action Items
담당	작업 내용	기한
...	...	...
---

**Few-shot의 목적**: AI에게 "이렇게 답변해야 한다"는 구체적인 패턴을 학습시킵니다.
