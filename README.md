# Codyssey B1-1: AI 프롬프트 엔지니어링 미션

**"ChatGPT한테 그냥 물어보는 것"에서 "제대로 업무 파트너로 시키는 것"으로 업그레이드하기**

## 📌 미션 개요
- **학습 기간**: 2026년 6월 ~
- **총 목표 시간**: 약 40시간
- **선택한 업무 과업**: **회의록 요약**  
  (회의 내용 → 결정사항, Action Items, 리스크, 후속 일정 등을 체계적으로 정리)

## 🎯 과제 목표
이 미션을 완료하면 다음을 설명할 수 있습니다:
- LLM의 확률적 생성 방식과 환각(Hallucination) 발생 원리
- Zero-shot / Few-shot / 단계적 추론(Chain-of-Thought) 기법의 차이와 적용 방법
- 평가 축 기반의 객관적 LLM 비교·선정 방법
- 페르소나와 시스템 프롬프트가 결과 품질에 미치는 영향
- 프롬프트 반복 개선 및 검증 프로세스

## 📁 Repository 구조
- `01_Model_Comparison/` — 모델 비교·선정 보고서
- `02_System_Design/` — 시스템 설계 문서 (페르소나, 프롬프트, Few-shot 등)
- `03_Execution_Logs/` — 10턴 이상 실행 로그
- `04_Hallucination_Test/` — 환각 검증

- `assets/` — 이미지 및 참고 자료
