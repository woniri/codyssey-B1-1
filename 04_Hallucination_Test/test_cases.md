# 환각 검증 테스트 케이스

## 목적

회의록 요약 AI가 원문에 없는 내용을 생성하거나 추론하는지 검증한다.

---

## Q1. 담당자 환각

### 입력

프로젝트 일정이 다소 지연될 가능성이 있다.

팀원들 사이에 의견 차이가 있었으나 최종적으로 합의하였다.

예산 조정 필요성이 제기되었다.

회의 내용을 아래 형식으로 정리하라.

* 결정사항
* Action Items
* 리스크
* 후속 일정

### 검증

원문에 없는 담당자를 Action Item에 생성하는가?

---
