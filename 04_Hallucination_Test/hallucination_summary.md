# GPT

## 특징

* 정보 부족 시 "결정된 사항 없음" 명시
* 원문 기반 응답 유지
* 추가 정보 생성 최소화

## 평가

환각 방지 성능이 가장 우수함.

---

# Claude

## 특징

* 형식을 유지하려는 경향 강함
* 빈 항목을 채우기 위해 내용을 생성함

## 확인된 환각 사례

* 후속 일정 생성
* Action Item 생성
* 검토 프로세스 생성
* 결정사항 확장

## 평가

환각 발생 빈도 높음.

---

# Gemini

## 특징

* 적극적인 업무 계획 생성

## 확인된 환각 사례

* 담당자 생성
* 예산 담당자 생성
* 후속 회의 생성
* 일정 생성
* 실행 계획 생성

## 평가

환각 발생 빈도 매우 높음.

---

# Grok

## 특징

* 형식 완성 중심 응답

## 확인된 환각 사례

* 예산 계획 생성
* 일정 조율 생성
* 대응 방안 생성
* 후속 일정 생성

## 평가

환각 발생 빈도 높음.

# 종합 분석

GPT는 원문에 없는 정보를 생성하지 않으려는 경향이 강했으며, 정보 부족 상황에서도 사실 기반 응답을 유지하였다.

반면 Claude, Gemini, Grok은 지정된 형식을 채우기 위해 원문에 존재하지 않는 Action Item, 후속 일정, 담당자, 실행 계획 등을 생성하는 사례가 확인되었다.

---

# 최종 결론

환각 검증과 문맥 유지 테스트를 종합 평가한 결과 GPT가 가장 안정적인 성능을 보였다.

특히 사실 기반 업무인 회의록 요약 과업에서 원문 충실도와 환각 방지 능력이 가장 우수하였다.

따라서 본 프로젝트의 최종 선정 모델은 GPT로 결정하였다.
