# 보너스 과제 1 - Gemini Gems 배포
(https://gemini.google.com/gem/1BxhhjBIHWw_U3YPwY2VwvimyOhBGhyZX?usp=sharing)

## 1. 개요

본 프로젝트에서는 최종 시스템 프롬프트를 재사용 가능한 형태로 활용하기 위해 Gemini Gems 기능을 이용하여 업무 전용 AI를 제작하였다.

생성된 Gem은 회의록 요약 업무에 특화된 AI로, 회의 내용을 입력하면 결정사항, Action Items, 리스크, 후속 일정을 자동으로 정리할 수 있도록 설계하였다.

---

## 2. Gem 정보

### Gem 이름

회의록 마스터 PM

### 목적

회의 내용을 구조화된 회의록 형태로 자동 요약

### 주요 기능

* 결정사항 추출
* Action Items 추출
* 리스크 식별
* 후속 일정 정리
* 실명 제거
* 환각(Hallucination) 최소화
* 정보 부족 시 확인 질문 수행

---

## 3. 적용 프롬프트

본 프로젝트에서 설계한 최종 시스템 프롬프트(v3)를 Gemini Gems의 Instructions에 적용하였다.

주요 규칙은 다음과 같다.

* 원문에 없는 내용 생성 금지
* 실명 대신 역할명 사용
* 정보 부족 시 확인 질문 수행
* 회의록 형식 강제
* 내부 추론 과정 비공개
* 환각 방지 검증 수행

---

## 4. 테스트 내용

### 테스트 입력

회의록 요약을 수행해줘.

출력 형식

* 결정사항
* Action Items
* 리스크
* 후속 일정

실명 사용 금지.
추측 금지.

회의 내용:

신제품 런칭 일정을 7월 15일로 확정하였다.
QA 리드는 성능 테스트 계획 수립이 필요하다고 제안하였다.
다음 회의는 6월 10일 오전 10시에 진행 예정이다.

### 테스트 결과

* 결정사항 추출 정상 수행
* Action Items 추출 정상 수행
* 후속 일정 추출 정상 수행
* 실명 사용 없음
* 원문 외 정보 생성 없음

---

## 5. 재사용성 평가

본 Gem은 특정 프로젝트에 종속되지 않으며 다양한 회의록 업무에 재사용할 수 있다.

사용자는 회의 메모, 회의록 초안, 음성 녹취록, 채팅 기록 등을 입력할 수 있으며 동일한 형식의 결과를 지속적으로 얻을 수 있다.

또한 시스템 프롬프트와 Few-shot 예시가 포함되어 있어 사용자 입력 방식이 달라지더라도 일정 수준 이상의 일관성을 유지할 수 있다.

---

## 6. 결론

Gemini Gems를 활용하여 회의록 요약 업무를 자동화하는 전용 AI를 구축하였다.

이를 통해 최종 프롬프트를 재사용 가능한 형태로 배포할 수 있었으며, 회의록 작성 업무의 생산성과 일관성을 향상시킬 수 있음을 확인하였다.
